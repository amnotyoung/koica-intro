# 핵심가치

KOICA는 지속가능발전목표(SDGs)의 5P와 맥락을 같이하는 다섯 가지 핵심가치를 바탕으로 일합니다.

1. **사람(People)** — 인간의 존엄성과 자유를 기반으로 사람 중심의 개발협력을 추진합니다.
2. **평화(Peace)** — 인권과 안전을 존중하며 지구촌 평화 정착에 기여합니다.
3. **번영(Prosperity)** — 협력국의 지속가능한 사회경제 발전과 동반성장을 지원합니다.
4. **환경(Planet)** — 탄소중립과 기후위기 대응을 포함한 지속가능한 삶의 터전 조성에 힘씁니다.
5. **파트너십(Partnership)** — 국내외 파트너와 협력해 개발협력 사업의 효과를 높입니다.
